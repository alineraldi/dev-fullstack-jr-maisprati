Contact Form With Validation

Instructions: Create a form with the following fields:

  • Name (required field)
  • Email (required field and must validate the email format)
  • Message (required field)
  • Phone (optional field)
  • Subject (dropdown field with 3 options: Question, Support, Feedback)
  • Checkbox to accept the Terms of Use (required)

Include a submit button that should only be enabled if all required fields are filled out correctly. Add a placeholder for each field. Use HTML5 to add validation for required fields and patterns (such as email and phone number fields).

CSS - Styling and Responsiveness:

  • Add a border around the form with rounded corners.
  • Use proper spacing between the fields. Change the background color and font style.
  • Apply focus to the fields, highlighting them when the user interacts with them.
  • Style the submit button to look like a modern button (with shadows, hover effect, and smooth animation when pressed).
  • Align the labels to the left and the input fields to the right to enhance readability.
